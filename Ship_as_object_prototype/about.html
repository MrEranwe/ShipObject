Ship as Object

The app recibes a text with the form as: 

{"ShipA":{"VesselData":{"Length":100, "Breadth":30, "Depth":8, "Draft":5, "BlockCoefficient":0.75, "PrismaticCoefficient":0.8},"ObjectsData":{"Engine":{"General":{"Type": "Engine", "Length":2, "Width":4, "Height":2}, "Capabilities": {"RPM": 500}}}, "HullData":{"Offsets":{"Data":[1, 2, 3]}}}}

and gives back the pretty form of the JSON element.

It calculates the total cost of the objects inside Objects Data and states which ones have no cost. It calculates the LCG due to the position and weight of the objets and plots them in a 3D model.
